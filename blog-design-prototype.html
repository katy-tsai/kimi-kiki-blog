1. Concurrent Features

React 18 引入了 Concurrent Features，這是 React 團隊多年來最重要的更新之一。這些功能讓 React 能夠同時準備多個版本的 UI，使得應用程式更加流暢...

javascript
import { useTransition } from 'react';

function App() {
  const [isPending, startTransition] = useTransition();
  
  const handleClick = () => {
    startTransition(() => {
      // 低優先級的更新
      setData(newData);
    });
  };
}
